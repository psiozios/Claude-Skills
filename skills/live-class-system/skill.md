# Live Class System

## Overview
Provides a comprehensive operational checklist for live event preparation — webinars, workshops, classes, and demos. Covers tech setup, content preparation, attendee communications, backup plans, and post-event follow-up. Ensures nothing is missed before going live.

## When to Use This Skill
- Preparing for a webinar or live workshop
- Setting up a recurring class series
- Planning a live product demo
- Any live event that requires coordination

## Process

### Pre-Event Checklist (1 week before)

**Content:**
- [ ] Slides/materials finalized
- [ ] Speaker notes complete
- [ ] Demo environment prepared and tested
- [ ] Backup slides (PDF) in case of tech failure
- [ ] Handouts/resources ready for distribution

**Tech:**
- [ ] Platform tested (Zoom, Teams, etc.)
- [ ] Screen sharing verified
- [ ] Audio/video quality checked
- [ ] Recording settings configured
- [ ] Backup internet option available
- [ ] Chat/Q&A settings configured

**Communications:**
- [ ] Registration confirmation sent
- [ ] Calendar invite with join link
- [ ] Reminder email scheduled (24h and 1h before)
- [ ] Post-event email draft ready

**Logistics:**
- [ ] Start/end times confirmed across time zones
- [ ] Co-host/moderator briefed
- [ ] Q&A process defined
- [ ] Breakout room setup (if applicable)

### Day-Of Checklist (30 min before)

- [ ] Join 15 minutes early
- [ ] Test audio and video
- [ ] Open slides/demo
- [ ] Start recording
- [ ] Welcome slide displayed
- [ ] Chat moderator ready
- [ ] Water and notes nearby
- [ ] Phone on silent

### Post-Event (within 24 hours)

- [ ] Stop recording and save
- [ ] Send follow-up email with recording link
- [ ] Share resources/materials
- [ ] Collect feedback (survey link)
- [ ] Update attendee list/CRM
- [ ] Note improvements for next time

## Key Rules
- Always have a backup plan for tech failures
- Test everything the day before AND 30 minutes before
- Send at least 2 reminder emails (24h and 1h before)
- Record everything — no-shows will watch the replay
- Debrief within 24 hours while memory is fresh
