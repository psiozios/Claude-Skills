# Substack Notes

## Overview
Creates high-performing short-form social content for Substack Notes using proven structural patterns while maintaining consistent voice. Optimizes for engagement and subscriber conversion.

## When to Use This Skill
- Promoting a new newsletter issue
- Building audience between issues
- Testing ideas before committing to full posts
- Engaging with the Substack community

## Process

### Step 1: Select Note Type
- **Teaser**: Preview upcoming content with a hook
- **Standalone Insight**: One valuable idea, fully delivered
- **Question/Poll**: Engagement-driven conversation starter
- **Behind the Scenes**: Process or personal reflection
- **Curated Link**: Add unique commentary to shared content

### Step 2: Apply Structure Pattern

**Hook Patterns (choose one):**
- Contrarian opener: "Most people think X. They're wrong."
- Curiosity gap: "I discovered something about X that changed how I..."
- Bold claim: "The single biggest mistake in X is..."
- Story lead: "Last week I [experience] and realized..."
- Data lead: "[Surprising statistic] — here's what it means"

**Body (2-4 short paragraphs):**
- One idea per paragraph
- Max 3 sentences per paragraph
- Use line breaks liberally
- Include one specific example or data point

**Close (choose one):**
- Question to drive comments
- Soft CTA to newsletter
- Provocative statement
- Actionable takeaway

### Step 3: Optimize
- Keep under 280 words (ideal: 150-200)
- Front-load value — the first line must stand alone
- No hashtags (Substack Notes doesn't use them)
- Add a relevant image or chart if available

## Key Rules
- Write like you talk, not like you write
- One idea per Note — never cram multiple topics
- The first line is the only line that matters for reach
- Ask questions that people can answer from experience
- Never hard-sell — offer value, let curiosity do the work
