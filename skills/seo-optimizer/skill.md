# SEO Optimizer

## Overview
Analyzes content for SEO opportunities, researches relevant keywords, and generates optimization reports. Produces an SEO-improved version covering title tags, meta descriptions, heading structure, keyword placement, and internal linking.

## When to Use This Skill
- Before publishing any web content
- Updating older posts for better search performance
- Planning new content around search intent
- Auditing existing content library

## Process

### Step 1: Content Audit
Analyze the current piece for:
- Primary topic and search intent (informational, navigational, transactional)
- Current heading structure (H1-H4)
- Word count and content depth
- Existing keyword usage (natural vs forced)

### Step 2: Keyword Research
Identify target keywords:
- **Primary keyword**: The main search term (1 phrase)
- **Secondary keywords**: 3-5 related terms
- **Long-tail variations**: 5-10 question-based or specific phrases
- **Semantic relatives**: Related concepts for topical authority

### Step 3: Optimization Report
Generate a report covering:
- **Title tag**: Under 60 characters, primary keyword near front
- **Meta description**: 150-160 characters, includes CTA
- **H1**: One per page, includes primary keyword naturally
- **Heading hierarchy**: Logical H2/H3 structure with keyword variations
- **Keyword density**: 1-2% for primary, natural placement throughout
- **Internal links**: 3-5 relevant internal link opportunities
- **Image alt text**: Descriptive, keyword-inclusive where natural
- **Readability**: Flesch reading ease score and improvements

### Step 4: Produce Optimized Version
Rewrite the content with optimizations applied, preserving the original voice and adding value rather than just stuffing keywords.

## Key Rules
- Never sacrifice readability for SEO — Google rewards user experience
- Write for humans first, search engines second
- One primary keyword per page — don't compete with yourself
- Front-load important keywords in titles and first paragraphs
- Internal linking should feel helpful, not forced
- Update meta descriptions to include a reason to click
