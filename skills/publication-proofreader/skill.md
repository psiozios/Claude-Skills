# Publication Proofreader

## Overview
Performs final technical polish before publishing. Checks grammar, punctuation, formatting consistency, broken links, image references, and generates 3-5 alternative headlines with A/B testing rationale.

## When to Use This Skill
- Final step before hitting publish
- After all content and structural edits are complete
- When you need headline alternatives for testing

## Process

### Step 1: Technical Proofread
- **Grammar and punctuation**: Catch errors without changing voice
- **Spelling**: Including proper nouns and brand names
- **Formatting consistency**: Bullet style, heading capitalization, date formats
- **Link verification**: Check all URLs are valid and point to intended destinations
- **Image references**: Verify all images are referenced and alt text exists
- **Markdown/HTML**: Validate rendering (unclosed tags, broken formatting)

### Step 2: Style Consistency
- Serial comma usage (pick one, be consistent)
- Number formatting (spelled out vs digits)
- Capitalization rules for product names, titles
- Quote style (smart quotes vs straight)
- Dash usage (em dash vs en dash vs hyphen)

### Step 3: Headline Alternatives
Generate 3-5 alternative headlines:
- **Curiosity-driven**: Creates an information gap
- **Benefit-led**: Promises a specific outcome
- **Data-driven**: Leads with a number or statistic
- **Contrarian**: Challenges conventional wisdom
- **Simple/Direct**: Clear and unambiguous

For each, provide:
- The headline
- Why it might outperform the original
- Best platform/context for this version

### Step 4: Final Checklist
- [ ] No typos or grammar errors
- [ ] All links working
- [ ] Formatting renders correctly
- [ ] Headline is strong
- [ ] Meta description written
- [ ] Author attribution correct

## Key Rules
- Preserve the author's voice — fix errors, don't rewrite
- Flag ambiguities but don't resolve them without asking
- Check the first and last paragraphs twice (most-read sections)
- Headlines should be testable — provide genuinely different approaches
