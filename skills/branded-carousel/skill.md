# Branded Carousel

## Overview
Transforms content into branded carousel posts for social media (LinkedIn, Instagram). Applies brand colors, fonts, and layouts consistently across slides. Structures content with hooks, value points, and CTAs. Outputs HTML designs ready for screenshotting or direct use.

## When to Use This Skill
- Repurposing blog posts or newsletters into carousel format
- Creating educational or how-to social content
- Building a consistent visual content series
- Promoting products or features visually

## Process

### Step 1: Content Extraction
From source material, extract:
- One compelling headline/hook
- 5-8 key points (one per slide)
- Supporting data or examples
- A closing CTA

### Step 2: Slide Structure

**Slide 1 (Hook):**
- Bold headline that stops scrolling
- Subtext that promises value
- Brand logo (small, corner)

**Slides 2-7 (Content):**
- One point per slide
- Large, readable text (minimum 24pt equivalent)
- Supporting icon or simple visual
- Consistent layout across all slides
- Slide number indicator

**Final Slide (CTA):**
- Clear call to action
- Profile tag or handle
- "Save this" or "Share" prompt

### Step 3: Visual Design
Apply brand system:
- Background color from brand palette
- Heading font for key text
- Accent color for emphasis
- Consistent padding and margins
- Brand-appropriate imagery style

### Step 4: Output
Generate as:
- Self-contained HTML (screenshot each slide)
- Individual slide images (if image generation available)
- Markdown version for content review

### Platform Specs
- **LinkedIn**: 1080x1080px or 1080x1350px, PDF upload, max 300 slides
- **Instagram**: 1080x1080px or 1080x1350px, max 10 slides

## Key Rules
- Maximum 15 words per slide — less is more
- Every slide must be readable in 3 seconds
- Consistent visual style across all slides (don't vary layouts)
- First slide determines if people swipe — invest time there
- Last slide must have a clear, single CTA
